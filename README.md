# VISA Status Checker

A bot created with Python, Playwright, SeleniumBase and Browserless.io to automatically check the status of a non-immigrant visa application at the Consular Electronic Appointment Center (CEAC).

Download the files, set up your tokens in Browserless.io (free) and Google Drive, click on play_bot.bat to activate it. Make sure to have your application details saved in a .txt file in the same folder so you can run it locally more easily.
